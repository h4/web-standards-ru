Новый встроенный в Firefox 15 отладчик теперь может соединяться для отладки кода на мобильных устройствах с [бетой Firefox Mobile][1] на Android, начиная с версии 2.2. В блоге разработчиков Mozilla сегодня были [представлены подробные инструкции и скринкаст][2], описывающие установку соединения между десктопным и мобильным Firefox. Впервые удалённая отладка появилась в Opera Dragonfly, уже поддерживается для Chrome Mobile и будет доступна в Mobile Safari, начиная с iOS 6.

[1]: https://play.google.com/store/apps/details?id=org.mozilla.firefox_beta
[2]: https://hacks.mozilla.org/2012/08/remote-debugging-on-firefox-for-android/
